## Engineering Principles

| Principle | Meaning |
|---|---|
| **Product first** | Build around real business workflows |
| **Server authority** | Important actions must be validated on the backend |
| **Security by default** | Auth, permissions, tenant isolation, and safe errors matter |
| **Clean UI** | Dashboards should be fast to understand and easy to use |
| **AI with control** | AI should assist, not bypass validation or permissions |
